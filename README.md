## 📌 Project Overview

Smart Healthcare AI Assistant is a C++ based system that helps users:
- Track health habits
- Monitor water and sleep
- Manage medicine reminders
- Analyze symptoms
- Get AI-style chatbot responses

---

## 🚀 Features

- 🔐 Login & Registration System  
- 💧 Water Tracker  
- 😴 Sleep Tracker  
- 💊 Medicine Reminder  
- 🤖 AI Chatbot (Keyword-based)  
- 🩺 Symptom Checker  
- 📊 Health Dashboard  
- 📈 Health Score System  
- 🔔 Notifications System  
- ⭐ Loading Effects UI  

---

## 🛠 Tech Stack

- C++
- File Handling (fstream)
- Loops & Conditionals
- String Processing
- Console-based UI
